# LG U+ 메타버스 아바타 빌더

> LG U+ 메타버스 사업의 3D 아바타 빌더 모듈. Unity 2020 기반 클라이언트로, 사용자가 헤어·얼굴·의상·악세서리를 조합해서 본인의 아바타를 만들고 마이룸에서 사진을 찍을 수 있는 기능을 담당. 5개월간 단일 컨트리뷰터로서 약 1/3 비중을 차지한 메인 개발자급 기여.

- **클라이언트**: LG U+ (메타버스플랫폼기술팀 협업)
- **활동 기간**: 2022년 5월 ~ 10월 (약 5개월)
- **역할**: Unity 클라이언트 메인 개발자

---

| **엔진** | Unity 2020.3 LTS · URP 10 |

## 3. 담당 도메인

### 3.1 아바타 객체 관리 시스템 (시그니처 영역)
아바타 파츠(헤어 · 얼굴 · 의상 · 악세서리 · 신발) 슬롯 기반 결합·교체 시스템.

- 모자 ↔ 헤어 충돌 처리
- 맨몸 아바타 폴백 시나리오
- 저장 직후 찰나 맨몸 로딩 이슈 해결
- Animator Rebind, LOD 거리 조절
- 단색 배경에서 캐릭터 사라짐 버그 수정

### 3.2 아바타 빌더 메인 UI
빌더 화면 전체 구성 — 카테고리 / 서브카테고리 / 색상 / 아이템 그리드.

- 무한 스크롤 그리드 컴포넌트
- 카테고리 탭 전환
- 성별 선택
- 배경 변경
- 아바타 포즈·제스처
- 로비 씬 진입

### 3.3 마이룸 (개인 공간)
- 마이룸 화면 구성
- **마이룸 사진찍기 기능 1차 완성**
- 아바타 스크린샷 캡처 → 썸네일 생성
- 단색 배경 + 캐릭터 합성, 썸네일 로딩 속도 개선

### 3.4 페이스 에디터
BlendShape 기반 얼굴 커스터마이징.

- 눈/코/입/턱 형태 슬라이더 편집
- 수염 BlendShape × 모자·헤어 결합 충돌 해결
- 모자 착용 시 컬러피커 적용 충돌 해결

### 3.5 컬러피커 시스템
UnityUI 기반 색상 피커 + 머티리얼 색상 실시간 반영.

- 모자 / 의상 / 헤어 컬러 변경
- BlendShape 결합 시 색상 충돌 일관화

### 3.6 에셋번들 & 다운로드 매니저
- **Addressables → AssetBundle 시스템 본인 단독 마이그레이션** ([대표 사례](#대표-사례-1-운영-방식에-맞춰-에셋-로딩-시스템-단독-마이그레이션))
- 다운로드 진행률, Android 외부 저장소 권한 처리
- 서버 키 기반 아바타 생성
- 동일 패킷 중복 송신 방지

### 3.7 서버 연동
- LG U+ 백엔드 API 통신 (REST)
- 아바타 데이터 저장 / 로드
- Android 플랫폼 판별 / 권한 부여

### 3.8 디버그·운영 편의
- 디버그 콘솔 단축키 토글
- 팝업 시스템 일관화
- lilToon 셰이더 제거 (성능·번들 사이즈 이슈)

## 대표 사례 1. 운영 방식에 맞춰 에셋 로딩 시스템 단독 마이그레이션

**Situation** — 프로젝트 초기 Addressables로 설계되었으나, LG U+ 운영 요구사항(번들 분할 / 다운로드 제어 / 캐싱 정책)에 부합하지 않는 부분이 있었음.

**Task** — AssetBundle 시스템으로 전환하면서 다운로드·캐싱·플랫폼 권한까지 안전하게 처리.

**Action**
- 5월 24일 Addressables 적용 → 5월 27일 다운로드 성공·Android 권한 부여 → 5월 30일 AssetBundle 시스템 전환 완료
- 다운로드 진행률 UI, 캐시 정책, 서버 키 기반 매칭, 동시 패킷 중복 송신 방지까지 함께 정리

**Result** — **1주일 내 인프라 교체 완수**. 이후 5개월간 안정적으로 운영.

## 대표 사례 2. 아바타 합성 엣지케이스 카탈로그 (2022-08)

**Situation** — 헤어·모자·수염·의상이 자유롭게 결합되는 시스템에서 사용자 시나리오에 따라 다양한 시각적 결함이 발생.

**Task** — 다대다 파츠 결합 시스템의 안정성을 운영 가능한 수준으로 끌어올리기.

**Action**
한 달 동안 84건의 작업으로 다음을 집중 해결:
- 모자 ↔ 헤어 ↔ 수염 결합 시 BlendShape 충돌
- 컬러피커가 BlendShape 영향 받는 파츠에서 누락
- 저장 직후 찰나 맨몸 로딩
- 단색 배경에서 캐릭터 사라짐
- 썸네일 로딩 속도

**Result** — 사용자 시나리오 기반 결함을 카탈로그화해서 잡아냄. 아바타 합성 시스템이 운영 단계에 진입.
