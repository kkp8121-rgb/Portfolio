# SlimeHunter (8-Bit Games 협업)

> 외부 게임사 8-Bit Games와의 협업 Unity 프로젝트. 익숙하지 않은 외부 코드베이스에 합류하여 **단일 영업일 내에 멀티샷 스킬 한 건을 깔끔하게 마무리**한 스팟 기여 사례. 짧지만 외부 코드 리딩 능력과 영향 범위 분석 능력을 보여주는 케이스.

- **협업 구조**: ovencode 측 + 8-Bit Games(외부) 공동 작업
- **활동 기간**: 2023년 8월 8일 (단일 작업일)
- **역할**: Unity 클라이언트 — 스팟 컨트리뷰터

## 3. 작업 내용 — 멀티샷(Multishot) 스킬

활(Bow) 무기 기반의 멀티샷 스킬을 신규 구현. 한 번 발사로 여러 발이 동시에 나가는 스킬이지만, 기존 발사체 9종 전체에 영향이 가는 구조였기 때문에 단순 신규 추가가 아니라 **공통 발사체 인터페이스 안전 확장** 작업이었음.

### 수정·신규 파일 (14개)

| 분류 | 내용 |
|------|------|
| **신규 스크립트** | 멀티샷 스킬 클래스 |
| **수정 스크립트** | 발사체 공통 모듈 보강 / 활 무기에 멀티샷 통합 |
| **신규 이펙트** | 멀티샷 전용 이펙트 프리팹 |
| **수정 프리팹** | 기존 발사체 9종 — 선인장탄·진화탄·계란·중력·호밍·나이프·바위·직선·수리검 |

## 4. 기여 포인트

### 4.1 외부 코드베이스 빠른 진입
8-Bit Games 주도의 익숙하지 않은 게임 아키텍처에 합류해서 **단일 영업일 내에 신규 스킬 한 건을 PR 완료**.

### 4.2 공통 발사체 모듈 안전 확장
멀티샷 도입 시 기존 발사체 9종(직선·중력·호밍·관통·바위 등)이 회귀하지 않도록 공통 모듈을 보강. **신규 기능이 기존 자산을 깨지 않게 하는 호환성 우선 접근**.

### 4.3 수직 슬라이스 단위 작업
스크립트(스킬·발사체·무기) + 프리팹(이펙트·9종 발사체) + 리소스 적용을 **한 작업 단위(Vertical Slice)** 로 마무리. 큰 코드베이스에서 한 기능을 끝까지 책임지는 작업 능력을 보여줌.
